# Конфигурация бота

## Файл config.json

В корне проекта находится файл `config.json`, который позволяет указать основные настройки бота без необходимости вводить их через веб-панель после каждого перезапуска.

### Настройки

Откройте файл `config.json` и заполните следующие поля:

```json
{
  "telegram_bot_token": "ваш_токен_бота",
  "telegram_bot_username": "имя_бота_без_собачки",
  "admin_telegram_id": "ваш_telegram_id"
}
```

### Важные замечания:

1. **telegram_bot_token** - токен вашего Telegram бота, полученный от @BotFather
2. **telegram_bot_username** - имя бота БЕЗ символа @ (например, `my_bot` вместо `@my_bot`)
3. **admin_telegram_id** - ваш Telegram ID (можно узнать у @userinfobot)

### Как это работает:

- При первом запуске бота настройки из `config.json` автоматически загружаются в базу данных
- Если настройки уже есть в базе данных, они не будут перезаписаны
- Вы можете изменить настройки через веб-панель, и они сохранятся в базе данных
- При следующем запуске бота настройки будут загружены из базы данных

### Пример:

```json
{
  "telegram_bot_token": "1234567890:ABCdefGHIjklMNOpqrsTUVwxyz",
  "telegram_bot_username": "my_vpn_bot",
  "admin_telegram_id": "123456789"
}
```
